\subsection{词法分析}

本研究选用的是开源的Jieba分词。目前主流分词工具通常包含了词性标注功能。Jieba分词完成词性标注的过程如下：

\begin{verbatim}
>>> import jieba.posseg as pseg
>>> words = pseg.cut(u'我爱北京天安门')
>>> for w in words:
...   print w.word, w.flag
...
我 r
爱 v
北京 ns
天安门 ns
\end{verbatim}
